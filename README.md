# Luxury Dhow Escapes

## Project info

A premium yacht charter and dhow cruise booking application for Dubai.

## Getting Started

To get started with this project locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
